Vim Important Mapping
======================

  | keys      |  Informations                             |
  |-----------|:------------------------------------------|
  | jk        | esc key                                   |  
  | ;         | : in insert mode                          |  
  | ,         | leader key                                |  
  | Space     | to toggle fold                            |
  | K         | to Ag / :Find on a word in normal mode    |  
  | \         | maps to :Find                             |  
  | ,t        | :FZF                                      |  
  | ,b        | :Buffers                                  |  
  | ,s        | :GitStatus                                |  
  | ,w        | :Gw                                       |
  | ,r        | :%s live search on word                   |
  | >         | indent on visual mode                     |
